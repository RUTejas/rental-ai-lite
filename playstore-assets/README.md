# RentWise Lite — Play Store asset checklist

This folder is for release artwork and listing copy. Do not place signing keys here.

- App icon: `app-icon-512.png` is provided at exactly 512×512 PNG. Review it at small launcher sizes before release.
- Feature graphic: manually create a 1024×500 PNG/JPG using the same forest, cream, and gold visual system. Do not stretch the square icon to fill it.
- Screenshots: capture real, non-demo-sensitive screens on Android. Include the landing/login flow, tenant dashboard, owner dashboard, rental records, and one Master Admin screen only if it contains no real personal data.
- Short description: maximum length is controlled by Play Console; suggested copy: “Rental records, bills, documents, and role-based dashboards in one clear app.”
- Full description: explain only features that are currently working. Do not claim payment processing, legal advice, exact install counts, or offline rental data.
- Privacy policy URL: `https://rental-ai-lite.vercel.app/privacy-policy`
- Contact email: set a real monitored address in `NEXT_PUBLIC_SUPPORT_EMAIL` and use the same address in Play Console.
- Suggested category: **Business** (alternative: Productivity).

Still required manually: 1024×500 feature graphic, final phone/tablet screenshots, reviewed listing copy, content rating, Data safety answers, target audience, support email, and final legal review.
